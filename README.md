# 🌴 Sawit Idle Clicker

Sawit Idle Clicker adalah game incremental (idle clicker) ringan berbasis web dengan tema perkebunan kelapa sawit Nusantara. Mulailah usahamu dengan memanen sawit secara manual, merekrut mandor, membangun pabrik, membeli sertifikat lahan, dan raih kekayaan sebagai juragan sawit!

## ✨ Fitur Utama

- **Klik & Panen Pasif**
  Dapatkan "Sawit per Klik" (SPC) dengan mengklik gambar sawit, dan tingkatkan produksi "Sawit per Detik" (SPS) dengan membeli berbagai macam *upgrade*.
- **Sistem *Upgrade* Produksi**
  Tersedia berbagai pilihan investasi mulai dari peralatan tradisional, tenaga kerja, hingga alat berat dan fasilitas canggih:
  - Klik Tangan (Meningkatkan SPC)
  - Egrek
  - Mandor
  - Truk Traktor
  - Gudang
  - Pabrik
- **Sistem Prestise (Sertifikat Lahan)**
  Gunakan hasil panen sawit untuk membeli *Sertifikat Lahan*. Setiap pembelian sertifikat akan memberikan bonus tambahan secara permanen berupa pengganda (multiplier) yang melipatgandakan seluruh hasil produksimu.
- **Pencapaian (*Achievements*)**
  Tantang dirimu dengan membuka berbagai pencapaian (achievements) seru. Setiap misi yang berhasil kamu selesaikan akan memberikan hadiah tambahan secara cuma-cuma seperti suntikan Sawit instan, bonus SPS, hingga peningkatan *multiplier*.
- **Penghasilan *Offline***
  Pabrik akan terus beroperasi meskipun kamu sedang menutup browser! Sistem permainan akan menyimpan progres offline kamu (maksimal hingga limit 4 jam) dan langsung memanennya ketika kamu kembali.
- ***Auto-Save* (Simpan Otomatis)**
  Jangan takut kehilangan progres. Data permainan otomatis tersimpan di *Local Storage browser* setiap 8 detik.

## 🛠️ Teknologi yang Digunakan

Proyek ini sepenuhnya dibangun secara komprehensif menggunakan sistem *stack* murni (Vanilla Web), membuatnya sangat ringan dan mulus tanpa memerlukan *build-tools*.
- **HTML5** (Semantik web)
- **CSS3** (Desain responsif, Layouting modern CSS Grid/Flexbox, retro font)
- **Vanilla JavaScript** (*Game Loop Logic*, Manipulasi DOM, API *Local Storage HTML5*)

## 🚀 Cara Memainkan

Sangat mudah untuk menjalankan game ini di perangkat manapun:

1. Unduh (Download ZIP) atau *clone repository* ini ke dalam komputermu.
2. Buka folder aplikasi `SawitIdleGame`.
3. Klik dua kali pada file `index.html` untuk menjalankannya langsung di peramban web (*browser* seperti Google Chrome, Mozilla Firefox, dll), 
   *Atau*
   Gunakan ekstensi seperti **Live Server** di VS Code jika ingin melakukan modifikasi secara *real-time*.
4. **Mulai bermain:** Silakan klik sawit, beli peningkatan, dan jadilah pengusaha kelapa sawit terkaya!

## ⚙️ Kontrol & Antarmuka

- **Menjalankan Game:** Klik kiri pada gambar Tandan Buah Sawit besar di layar.
- **Volume:** Tombol 🔊/🔇 berada pada pojok kanan atas Header untuk mengatur efek suara *game*.
- **Multi-Pembelian:** Terdapat tombol *toggle* pada bagian atas daftar *upgrade* (`x1`, `x10`, `Maks`) untuk berbelanja *upgrade* dalam jumlah banyak secara massal.
- **Statistika & Panel Pemantauan:** Pantau seluruh statistik kamu (Total klik, Waktu Main, Saldo) dan progres misi pencapaian (*Achievements*) pada layar samping.

## 📝 Lisensi

Proyek permainan `Sawit Idle Clicker` bersifat *open-source*. Silakan digunakan untuk keperluan belajar membuat *idle game* sederhana berbasis JavaScript, menyalin *codebase*-nya, atau memodifikasinya menjadi game *clicker* bertema impianmu.
